### Tecnologias

[Descarga Visual Studio Code](https://code.visualstudio.com/)

-Para el Backend:

Debes tener instalado python 3.9.8: 
[Descarga python](https://www.python.org/downloads/windows/)

Debes tener instalado ionic:
